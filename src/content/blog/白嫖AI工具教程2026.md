# 🚀 白嫖 AI 工具全攻略 - 2026 版

## 一、阿里 iFlow CLI（免费无限用 Kimi/MiniMax）

### 1.1 安装

**macOS/Linux:**
```bash
# 方式一：npm（推荐）
npm install -g @ali-iflow/iflow-cli

# 方式二：curl 一键安装
curl -sSf https://iflow.cn/install | sh
```

**Windows:**
```bash
# 先安装 Node.js，然后
npm install -g @ali-iflow/iflow-cli
```

### 1.2 登录

```bash
# 启动并登录
iflow auth login
```

会弹出浏览器，用支付宝/淘宝账号扫码登录即可。

### 1.3 使用方法

```bash
# 直接对话（默认模型）
iflow "帮我写一个 Python 脚本"

# 指定模型
iflow -m kimi-k2.5 "用中文解释什么是量子计算"
iflow -m minimax-m2.1 "写一首关于春天的诗"

# 查看可用模型
iflow models list
```

### 1.4 进阶：配置到 VS Code / Cursor

在 Cursor 设置中找到 AI 配置，填入 iFlow 提供的 API Endpoint 和 Key。

## 二、GitHub Models（每天 150 次免费 Claude/GPT）

### 2.1 获取访问权限

1. 打开 https://github.com/marketplace/models
2. 点击 "Get started" 或 "Join waitlist"（现在基本已开放）
3. 用 GitHub 账号授权

### 2.2 获取 API Key

1. 进入 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 权限勾选：`repo`、`read:user`
4. 生成后复制 Token

### 2.3 使用方法

**方式一：Web Playground**
直接访问 https://github.com/marketplace/models，点击 "Try in Playground" 即可在线使用。

**方式二：cURL**
```bash
export GITHUB_TOKEN="你的_token"

curl -X POST "https://models.inference.ai.azure.com/chat/completions" \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "用中文写一首诗"}],
    "temperature": 0.7
  }'
```

**方式三：Python**
```python
import requests

url = "https://models.inference.ai.azure.com/chat/completions"
headers = {
    "Authorization": f"Bearer {os.getenv('GITHUB_TOKEN')}",
    "Content-Type": "application/json"
}
data = {
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "你好"}]
}
response = requests.post(url, headers=headers, json=data)
print(response.json())
```

## 四、常见问题

**Q: iFlow 登录失败？**
A: 确保用支付宝/淘宝账号扫码，或者尝试 `iflow auth logout` 后重新登录。

**Q: GitHub Token 权限不足？**
A: 确保 Token 有 `repo` 和 `read:user` 权限。

**Q: 每天 150 次不够用？**
A: iFlow 无限量，可以主要用 iFlow，GitHub 做备用。

**Q: 国内访问慢？**
A: iFlow 国内快；GitHub 可能需要代理。
